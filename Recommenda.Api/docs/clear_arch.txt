* Domínio (Domain) - O Coração

Onde residem as Entidades e Regras de Negócio puras.

* Aplicação (Application) - O Cérebro

Contém os Casos de Uso (Use Cases) da aplicação. Ela coordena o fluxo de dados.

* Infraestrutura (Infrastructure) - As Ferramentas

Implementa as interfaces definidas no Domínio ou Aplicação.

* Apresentação / API - A porta de entrada do sistema

Como o usuário interage com o sistema.
